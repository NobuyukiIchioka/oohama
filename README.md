# 配信方法

1. node js 18 をインストール
2. README.mdがあるディレクトリに移動して初期化

```sh
$ npm install
```

3. 本番ビルド

```sh
$ npm run build
```

4. 配置

buildディレクトリ配下のファイルをサーバーに配置する。
